# Application Estimation Devis

Application web de simulation et génération de devis professionnels pour 2S Construction.

## Description

Cet outil permet de :
- Générer des devis BTP automatiquement
- Estimer les coûts de main d'oeuvre et matériaux
- Exporter les devis en PDF
- Sauvegarder les prix standards

## Technologies

- React + TypeScript
- Vite
- Supabase (base de données)
- Tailwind CSS

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvrir http://localhost:5173
